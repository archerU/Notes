# CPV

CPV 类目属性值

C是类目集合，P是属性项集合，V是属性值集合

- 一个类目上的关键属性即可决定一个 SPU
- SPU=关键属性+商品属性+普通属性+销售属性。
- SKU = SPU + 价格 +库存。


```
名词	解释
CVR	Conversion Rate，点击转化率=购买量/点击量
CTR	点击率，即每次展现带来的点击数量--p4p 流量下的点击率
CTR1	点击率，即每次展现带来的点击数量--主搜流量下的点击率
CTRi	点击率，即每次展现带来的点击数量--单个宝贝的点击率
DAU	该场景每日访问的UV数，用于表达每日用户活跃度
DPV	该场景引导的1跳点击PV+2跳点击PV
GMV	Gross Merchandise Volume，成交总额
IPV	该场景引导至ipv页面的PV数，ipv页面：item或spu或shop页面
MAU	该场景每月访问的UV数，用于表达每月用户活跃度
PV	Page View，代表页面访问量
PCTR	点击PV/展示PV，Click-Through-Rate，用于衡量坑位级别的点击效率
PVR	一个业务的曝光PV/信息流总曝光PV，用来衡量一个业务在信息流的曝光占比
PPC	点击单价，即每次点击产生的消耗，单位通常为元
UV	Unique Visitor，访问人数
UCTR	点击UV/展现UV，用于衡量用户在该场景的转化程度(反向指标跳失率=1-UVctr)
UV价值	成交金额/访问UV，每个访问用户带来的平均成交金额
成交转化率	成交UV/访问UV，来了多少人，有多少人买
次日留存	当天来访问的用户中，第二天还会再来的比例，衡量用户对一个产品的粘性
货单价	成交金额/成交件数
客单价	成交金额/成交UV，成交用户的平均成交金额
7日留存	当前来访问的用户中，之后7天内还会再来的比例，衡量用户对一个产品的粘性
千展成交	成交/曝光PV*1000，用于衡量信息流千次展现带来的成交金额
千展DPV	DPV/曝光PV*1000，用来衡量每个业务在用户端的吸引力价值
人均曝光PV	场景曝光PV/场景曝光UV，用于衡量用户的流量深度
人均DPV	DPV/场景点击UV，用于衡量每个用户在信息流的点击次数
takerate	客户广告支出占同期店铺收入的比例
有效vv	用来衡量内容(视频/图文)是否被用户有效的消费
```
