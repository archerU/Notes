# 我对数据分析和洞察的几点感悟

## 洞察的目的？

洞察是指看穿，看透，看到本质。

## 数据分析？

数据分析的流程，可以分成六步，分别是明确目的、找到数据、整理数据、分析数据、图形呈现、结果报告。

数据分析用途：现状分析、原因分析和预测分析。

数据分析三个方法：杜邦分析法、漏斗模型分析法、四象限/矩阵分析法。

数据分析方法论：PEST分析法、SWOT分析法、5W2H分析法、4P营销理论、AARRR模型
