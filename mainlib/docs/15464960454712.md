# Git

## Git 本地分支操作

删除本地分支
```
git branch -d <name>
```

## Git 远程分支

删除远程分支
```
git push origin -d <name>
```

## Git 修改远程仓库地址

查看远程仓库
```
git remote -v 
```

1、修改命令
```
git remote origin set-url [url]
```

2、先删后加
```
git remote rm origin
git remote add origin [url]
```

3、直接修改 config 文件
```
.git -> config
```
